1. 단일 if문

age가 5냐?
{% if age == 5 and 4 == 4 %} age가 5가 맞다. {% endif%}
age가 5가 아니냐?
{% if not age == 5 or 4 == 4%} age가 5가 아니다. {% endif%}

2. if ~ else 문

age가 15면 "맞다" 출력하고 아니면 "틀리다" 출력
{% if age == 15 %} age가 15가 맞다. {% else %} age가 15가 아니다. {% endif %}

3. 다중 if 문

username 길이가 5글자보다 많으면 "많다", 3글자보다 많으면 "중간" 모두 아니면 "적다"
{% if username|length >= 5 %} 많다 {% elif username|length >= 3 %} 중간 {% else %} 적다 {% endif %}
